## 物理现象

论文实验覆盖了12种日常生活与物理场景中常见的运动/形变现象，具体包括：

1. 匀速运动（Uniform Motion）
2. 匀加速运动（Acceleration）
3. 匀减速运动（Deceleration）
4. 抛物运动（Parabolic Motion）
5. 3D运动（3D Motion）
6. 斜面滑动（Slope Sliding）
7. 圆周运动（Circular Motion）
8. 定轴转动（Rotation）
9. 带自转的抛物运动（Parabolic Motion with Rotation）
10. 阻尼振动（Damped Oscillation）
11. 尺寸变化（Size Changing）
12. 形变（Deformation）

**物理参数**
实验的物理参数分为**通用参数范围**和**各运动类型的核心物理量**两类，具体如下：

- **通用物理参数范围**
    1. 速度量级：测试的运动速度大多在**0–15 m/s**区间内；
    2. 运动时长：生成视频中物理运动的持续时间集中在**1–2秒**；
    3. 训练数据规模：针对每种运动，从物理模拟器中采集**100个不同初始条件**的物理清洁视频作为训练数据；
    4. 像素-物理单位映射：将帧间像素变化转换为物理单位时，采用**0.00625米/像素**的换算系数；
    5. 预处理滤波：对所有视频采用**5帧移动平均滤波**以降低物理量估计的噪声。

- **各运动类型的核心物理量与控制参数**
    论文通过**物理不变性分数（PIS）**评估各运动的物理一致性，对应不同运动的关键物理量（即PIS计算的核心不变量）及部分测试/训练参数如下：
    | 物理现象          | 核心物理量（PIS评估指标）| 典型初始/控制参数（实验设定）|
    |-------------------|------------------------------------------|-----------------------------------------------------------------------------|
    | 匀速运动          | 水平速度$v_x$                            | 如金属立方体/保龄球沿平直表面匀速滑动，初始速度保持恒定                     |
    | 匀加速/减速运动   | 水平加速度$a_x$（减速为$-a_x$）| 如SUV沙地匀加速、跑车乡村道路匀减速，加速度大小保持稳定                     |
    | 抛物运动          | 水平速度$v_x$、竖直加速度$a_y$（重力加速度） | 篮球/棒球斜抛，水平方向无加速度，竖直方向加速度近似恒定重力加速度           |
    | 3D运动            | 竖直速度$v_y$、长轴尺寸增量率$\Delta l$   | 物体向镜头移动（尺寸随距离减小而增大），$v_y$保持恒定，$\Delta l$变化均匀   |
    | 斜面滑动          | 水平加速度$a_x$、竖直加速度$a_y$         | 手机/书本沿光滑斜面下滑，忽略摩擦时加速度分量恒定                           |
    | 圆周运动          | 轨道角速度$\omega$                       | 小天体绕恒星公转，$\omega$保持恒定（如初始角速度设定为1.0 rad/s，见Fig.2(b)） |
    | 定轴转动          | 自转角速度$\omega$                       | 木杆/金属杆绕自身轴线转动，$\omega$均匀无突变                               |
    | 带自转的抛物运动  | $v_x$、$a_y$、自转角速度$\omega$          | 画笔/钢笔斜抛并自转，同时满足抛物轨迹和匀速自转规律                         |
    | 阻尼振动          | 竖直加速度$a_y$                          | 悬挂铃铛/单摆小幅摆动，$a_y$随摆角呈周期性变化且幅值衰减                   |
    | 尺寸变化          | 半径增量率$\Delta r$                     | 气球充气时半径均匀增大，$\Delta r$保持恒定（如初始尺寸0.3–0.6，见Fig.31）|
    | 形变              | 长轴尺寸增量率$\Delta l$                  | 面团/酸奶摊平过程中，长轴尺寸均匀延展，$\Delta l$变化稳定                   |

此外，在NND模块训练中，还定义了**9维潜物理状态向量**$Z=[x,y,v_x,v_y,\theta,\omega,s,l,a]$，其中各分量的物理含义为：$x/y$为质心位置、$v_x/v_y$为质心速度、$\theta$为旋转角度、$\omega$为角速度、$s/l$为最短/最长维度、$a$为投影面积，该向量为所有运动的参数化提供了统一表征。
